📌 Personal Portfolio Website
🧠 Overview

This project is a personal portfolio website developed to showcase my skills, projects, and contact information in a structured and interactive manner. It serves as a digital representation of my work and technical abilities.

🎯 Features
👤 About section with personal introduction
🧠 Skills section with visual progress bars
💼 Dynamic projects section (rendered using JavaScript)
🔗 Social media links (GitHub, LinkedIn, LeetCode, Email)
📬 Contact form with basic validation
🎯 Smooth navigation between sections
⚙️ Technologies Used
HTML
CSS
JavaScript
🧩 How It Works
The website layout is created using HTML and styled using CSS
JavaScript is used to dynamically display projects
Event listeners handle user interactions like button clicks and form submission
External links connect to GitHub, LinkedIn, and other platforms
🧠 Concepts Used
DOM Manipulation
Event Handling
Arrays and Objects
Promises (for simulating async behavior)
Form Validation
Basic UI structuring
🚀 Future Enhancements
Responsive design for mobile devices
Advanced animations and transitions
Dark/Light theme toggle
Project filtering system
Resume download feature
🎯 Purpose

This project demonstrates my ability to build a structured web application using core web technologies and implement interactive features using JavaScript.
