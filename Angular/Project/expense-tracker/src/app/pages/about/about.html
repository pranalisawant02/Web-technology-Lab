About Expense Tracker

Expense Tracker is a simple Angular application used to record and monitor daily expenses. The user can add expenses, delete expenses, view total spending, and see category-wise expense distribution using a pie chart.

Angular Concepts Used

Concept Short Definition Where Used
Component A reusable UI block in Angular. Home component and About component.
Routing Used to navigate between different pages. Home page and About page navigation.
Service Used to write reusable business logic. ExpenseService handles add, delete, get, and total calculation.
Interface Defines the structure of data. Expense interface defines id, title, amount, category, and date.
Two-way Binding Connects form input and TypeScript variable together. Used with [(ngModel)] in the expense form.
Event Binding Executes a function when user performs an action. Used in Add Expense form submit and Delete button click.
*ngFor Repeats HTML for every item in an array. Used to display all expenses.
*ngIf Shows or hides HTML based on condition. Used to show empty message or chart/list.
LocalStorage Stores data inside browser memory. Used to save expenses after page refresh.
Lifecycle Hook Special method that runs at a specific component stage. ngAfterViewInit is used to create the chart after HTML loads.
