# 📌 Fake Exam Proctor

## 🧠 Overview

Fake Exam Proctor is a web-based application that simulates an online exam monitoring system. It tracks user behavior during an exam and detects suspicious activities such as tab switching, inactivity, and unusual interactions.

---

## 🎯 Features

* 👁️ Detects tab switching using Page Visibility API
* ⏳ Tracks user inactivity
* 🖱️ Monitors user interactions (clicks/events)
* ⚠️ Displays warnings for suspicious behavior
* 🔒 Option to disable exam after multiple violations

---

## ⚙️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 🧩 How It Works

1. User starts the exam
2. System begins monitoring activity
3. If user switches tab → warning triggered
4. If user remains inactive → alert displayed
5. After multiple violations → exam is restricted/disabled

---

## 🧠 Concepts Used

* DOM Manipulation
* Event Handling (`click`, `visibilitychange`)
* Timers (`setInterval`, `setTimeout`)
* Conditional Logic
* Browser APIs (Page Visibility API)
